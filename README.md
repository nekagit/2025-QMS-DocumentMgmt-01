# Document Management System - 2025-AIGenName-01

A Quality Management System application for managing documents with version control.

## Features

- Document creation and management
- Version control
- Category classification
- Status tracking
- Document listing and deletion

## Tech Stack

- **HTMX** - Dynamic HTML interactions
- **SCSS** - Styling
- **TypeScript** - Type-safe JavaScript
- **Express.js** - Server framework

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Usage

1. Start the server: `npm run dev`
2. Open your browser to `http://localhost:3001`
3. Add documents using the form
4. View and manage documents in the list
